# Cinema Page

Це верстка сайту для кінотеатру із адаптивним дизайном.

## 🚀 Технології:

- HTML
- SCSS
- JavaScript
- Gulp

### 📂 Структура проєкту:

/src - вихідні файли

    │── /assets

        │── /fonts - шрифти

        │── /icons - іконки

        │── /img - зображення

    │── /css - згенеровані CSS

    │── /js - скрипти

    │── /sass - стилі

    │── index.html - головна сторінка

│── gulpfile.js - конфігурація Gulp

│── package.json - налаштування npm
