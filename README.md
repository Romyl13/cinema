# Hotel Landing Page

Це верстка сайту для кінотеатру із адаптивним дизайном.

## 🚀 Технології:

- HTML
- CSS
- JavaScript
- Gulp (для автоматизації)

## 📂 Структура проєкту:

/src - вихідні файли

│── /sass - стилі

│── /css - згенеровані CSS

│── /js - скрипти

│── /img - зображення

│── index.html - головна сторінка

│── gulpfile.js - конфігурація Gulp

│── package.json - налаштування npm

│── dist - фінальна версія сайту
